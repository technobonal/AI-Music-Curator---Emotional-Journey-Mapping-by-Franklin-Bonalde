# AI Music Curator - Emotional Journey Mapping

## 🛠️ Resumen Técnico

- **Rol:** Music Consultant & Sound Architect
- **Técnica:** Structured Prompting / Emotional Progression Logic

## 💡 Descripción del Proyecto

Este repositorio contiene un prompt avanzado diseñado para controlar el comportamiento de una IA y generar curadurías musicales precisas. El objetivo es crear una experiencia auditiva basada en un viaje emocional progresivo, demostrando control de parámetros, estructuras temáticas y limitaciones de género.

**Ver el prompt:** Para ver el prompt completo que diseñé, abre el archivo [prompt.md](./prompt.md).

**Ver el resultado:** Puedes ver un ejemplo de la playlist generada por la IA en el archivo [output.md](./output.md).
